## form-generator JSON uniapp解析器

> 用于将form-generator导出的JSON解析成一个表单。

### 依赖

> 依赖于uview1.8组件库

### 安装组件

```
npm i uniapp-form-generator-parser
```

或者

```
yarn add uniapp-form-generator-parser
```
